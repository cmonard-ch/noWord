## Chapter block

| Key       |      Description      |
|:----------|:--------------------- |
| type      |  chapter              |
| title     |  Title of the chapter |
| level     |  chapter hierarchy level (0,1,2,3,...) |
| content   |  First paragraph      |

Example:
```YAML
- type    : chapter
  level   : 0
  title   : Some Chapter Title
  content : >
    Some text...
```
